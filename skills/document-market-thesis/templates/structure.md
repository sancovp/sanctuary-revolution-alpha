# Market Thesis Structure Quick Reference

## The 8 Files

```
new_market/
├── 00_overview.md          # Thesis + why now + file index
├── 01_death_of_X.md        # Old model failure modes
├── 02_Y_as_product.md      # What the new thing IS
├── 03_Z_progression.md     # How users "win"
├── 04_economics.md         # Cost structures, viability
├── 05_your_story.md        # Journey, failures, pivot
├── 06_democratization.md   # Barriers → phases → vision
└── 07_complete_picture.md  # Stack + table + one-liner
```

## Per-File Templates

### 00_overview.md
```
# The New Market: [NAME]
## The Thesis (1 paragraph)
## Why Now (3-4 subsections)
## The Market Opportunity
## File Index
```

### 01_death_of_X.md
```
# The Death of [Old Model]
## The Old Model (show the loop)
## Why It Fails
  - The Transfer Problem
  - The Context Problem
  - The Motivation Problem
## The New Model (show the new loop)
## The Key Insight
## Completion Rates (old vs new)
```

### 02_Y_as_product.md
```
# [New Thing] as Product
## What Is [It]?
## The Layers (architecture diagram)
## Why [Base] Matters
## The Builder Pattern
## Higher Order Types
## The Product Implication
```

### 03_Z_progression.md
```
# [System]: The Progression System
## What Is [It]?
## The Game (progress visualization)
## The Dimensions (what's tracked)
## Why Self-Reported?
## The Course Structure
## The Win Condition
## Why This Changes Everything
```

### 07_complete_picture.md
```
# The Complete Picture: [Title]
## The Stack (ASCII diagram)
## The Flow (step diagram)
## Why This Is The Future (table)
## The One-Liner
## The Market Size
## Your Position
```

## The One-Liner Formula

> "[Product type] are [what users do] WITH [enabler] that result in [transformation]."

Example:
> "PAIA infoproducts are courses you do WITH an AI agent that result in you having a working AI agent."
