## KEY ARCHITECTURE

### CAVEAgent = Hypervisor
- Hooks live in Claude Code agents (paia_*.py in ~/.claude/hooks/)
- CAVEAgent receives signals from any agent's hooks
- CAVEAgent is the CONTROL LAYER for multiple agents
- Single runtime, multiple agents

### Loop Flow
```
start_loop("autopoiesis")
    ↓
1. Get AgentInferenceLoop from AVAILABLE_LOOPS
2. Write /tmp/hook_config.json (enables hooks)
3. Register handlers with HookRouterMixin
4. Run on_start() callback
    ↓
LOOP IS HOT - hooks fire, handlers run, context injected
    ↓
stop_loop() or conditions trigger
    ↓
1. Run on_stop() callback
2. Unregister handlers
3. Clear hook_config.json
```

### Self-Loading Pattern
The agent can call `curl -X POST localhost:8080/loops/start -d '{"loop": "autopoiesis"}'`
to load a loop ONTO ITSELF. Then operates under that loop until conditions.
