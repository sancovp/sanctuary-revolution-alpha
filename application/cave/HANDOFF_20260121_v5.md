## INTENT (the technical vision)

**Hooks AND Loops both reified in code:**

1. **Hook logic** = CodeAgentSDNAC with Ariadne chains
   - NOT "inject reminder strings"
   - The ACTUAL behavior (like brainhook's "block stop if enabled")

2. **Loop** = AgentInferenceLoop wrapping CodeAgentSDNAC
   - Configures which hooks are ON
   - The hook_ariadnes define what those hooks DO

3. **CAVEAgent loads loop** → hooks fire in main agent with that behavior
