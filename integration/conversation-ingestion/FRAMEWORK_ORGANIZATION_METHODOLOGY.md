## Semantic Search Technique for Framework Organization

**Discovery**: When facing confusion about framework structure, use conversation ingestion tools to search existing frameworks and conversations:

### The Pattern:

1. **Identify keywords** related to the question (roadmap, tier, mvp, phase, etc.)
2. **Search frameworks** by name/definition for those keywords
3. **Bundle tagged pairs** using `bundle_multi_tag` tool
4. **Read relevant pairs** to find actual design decisions
5. **Extract definitions** from emergent_frameworks.json
6. **Trace dependencies** through framework relationships

### Example from This Session:

```python
# Search for roadmap frameworks
keywords = ['roadmap', 'mvp', 'phase', 'launch', 'tier']
```
