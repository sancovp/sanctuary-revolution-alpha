# Meta-Composition (Empire of Empires)

> Compose category-level patterns into a pattern-of-patterns

## When to use
You've reached the CategoryKing level and want to go further — what governs multiple category kings? What's the pattern of being a conglomerate?

## Process

1. **Create the meta space**:
   ```
   crystal_ball(cmd: "create_space", body: { name: "EmpireOfEmpires" })
   ```

2. **Add the meta-pillars**:
   ```
   add_point → CategoryKing              ← The pattern itself becomes a node
   add_point → Cross_Category_Synergies  ← What happens between categories?
   add_point → Capital_Allocation         ← How resources flow across empires
   add_point → Meta_Brand                 ← The brand that contains brands
   add_point → Acquisition_Engine         ← How new categories get absorbed
   ```

## Key insight
At this level, `CategoryKing` is just a node — one child of `EmpireOfEmpires`.
The thing that was the whole world at the previous level is now just one component.

This is the universal scaling move:
**The container at level N becomes a component at level N+1.**
